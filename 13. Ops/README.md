# 13 · Ops / VPS

Utilidades para operar el VPS.

- `sshpass` — con fallback al tap `hudochenkov/sshpass` si no está en homebrew-core.

## Uso
```bash
./setup.sh 13
```

## Notas
- `sshpass` permite pasar la contraseña SSH de forma no interactiva; úsalo solo en entornos controlados (es menos seguro que claves SSH).
